HTML:Lenguaje de Contenido

El HTML(HyperText Markup Lenguage) es el lenguaje estandar para la creación de páginas web. Su función principal es estructurar y dar formato al contenido,permitiendo que los navegadores lo interpreten correctamente.

No es un lenguaje de programación

A menudo, los estudiantes confunden HTML con un lenguaje de programación. Sin embrago, HTML no permite ejecutar lógica, ni crear algoritmos, ni hacer operaciones matemáticas. Su objetivo es describir el contenido y cómo este se organizaen una página web.

El concepto de marcado

HTML funciona mediante etiquetaso tagsque rodean el contenido y le otorgan significado. Por ejemplo, la etiqueta <p>indica que el texto es un párrafo,mientras que indica un encabezado principal.

Estructura básica

Una página HTML,comienza con la declaración <DOCTYPE html>seguidas por las secciones <head> y <body>. En el headse define los metadatos, mientras que en el bodyse coloca en contenido visible.

El rol de HTML en la web

En el ecosistema web, HTML es el esqueleto. Luego se completan con CSSpara estilos y JavaScrippara la lógica y la interactividad.

Conclusión

Aprender HTML, es fundamental para cualquier estudiante de desarrollo web. Recordemos: HTML no es programación, si no un lenguaje de marcadoque define la estructura del contenido en la web.
